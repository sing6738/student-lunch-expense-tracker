# CHANGELOG - Lunch Expense App

บันทึกการเปลี่ยนแปลงและพัฒนาแอปพลิเคชัน

## [2026-06-17] - ปรับปรุง UI และเติมเต็มฟีเจอร์

### Added
- **ระบบจัดการร้านและเมนู**: เพิ่มหน้า `manage_menu.html` เพื่อให้ผู้ใช้สามารถ เพิ่ม/แก้ไข/ลบ ร้านอาหารและเมนูได้เองจากหน้าเว็บ
- **หน้า Error Pages (404, 500)**: เพิ่มหน้าที่ออกแบบมาสวยงามและเข้ากับธีมแอป เมื่อเกิดข้อผิดพลาด
- **Favicon**: เพิ่มไอคอน Lunch Box (🍱) บนแท็บเบราว์เซอร์
- **แอนิเมชั่น**: เพิ่มระบบ `animate-in` และ `staggered delays` ทำให้องค์ประกอบในหน้าเว็บเคลื่อนไหวเมื่อโหลด
- **ลูกเล่น UI**: เพิ่มเอฟเฟกต์ Glassmorphism, Hover glow และ Pulse animations

### Improved
- **Profile Page**: ปรับโฉมใหม่ให้ดูพรีเมียมขึ้น พร้อมแสดงสถิติระยะเวลาการเป็นสมาชิกและยอดรวม
- **Analytics**: ปรับปรุงกราฟให้สวยงามขึ้น ใช้ธีมสีม่วง และมีแอนิเมชั่นการวาดกราฟที่นุ่มนวล (Doughnut chart, Rounded bars)
- **Navigation**: เพิ่มลิงก์ "จัดการร้านและเมนู" ใน Sidebar และจัดระเบียบเมนูใหม่
- **CSS System**: เพิ่ม Utility classes เช่น `.text-accent`, `.glass-panel`, `.btn-glass` เพื่อความสะดวกในการพัฒนาต่อ

### Fixed
- ตรวจสอบความถูกต้องของเส้นทาง (Routes) ใน `app.py` และลงทะเบียน Error handlers
